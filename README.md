# PluginSystem

---

PluginSystem build with [deno-core](https://github.com/denoland/deno_core)

---

**Cargo.toml**

```toml
plugin-rs = { git = "https://github.com/blockz-dev/plugins-rs", branch = "dev" }
```

**Features**

+ Embeded Plugins
+ Simple Standalone Plugins
+ Archived Plugins with `.tar.xz`

**Not Supported yet**

+ npm, jsr packages (only local)
